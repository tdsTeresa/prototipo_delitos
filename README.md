
# Análisis de delitos municipales con Python y SQL
<h2>🟡 Descripción general: </h2>
<br>
Durante esta actividad Se llevó a cabo la conexión y la creación de objetos cursores en Python para realizar la transformación de datos  y consultas SQL y obtener un panorama general de los delitos en México durante los últimos años (2025). Se incorporó además, un reporte dinámico en Looker Studio enfocado a los crímenes relacionados con automóviles, teniendo en cuenta información que podría servir en la definición de precios para una aseguradora de autos.

<h2>📊 Actividades: </h2>
<br>
  • Creación de base de datos para alojar la tabla delitos.<br><br>
  • Conexión a la base de datos en Python mediante la librería pyodbc.<br><br>
  • Creación de objeto cursor.<br><br>
  • Transformación de datos.<br><br>
  • Definición de tablas.<br><br>
  • Operaciones de lectura (métodos execute, fetchone(), fetchall(), fetchmany()) e inserción (fast_executemany, executemany). <br><br>
  • Consultas a la base de datos mediante combinación de tablas (JOINS) y filtros (Distinct, Like, uso de CTE). <br><br>
  • Creación de archivo csv con filtros de los delitos relacionados con autos. <br><br>

<h2>🔵 Observaciones generales: </h2>
<br>
   • En 2025 de manera general, el promedio de delitos es más alto en municipios como León, Tijuana y Ecatepec de Morelos. 
<br><br>
  • El delito más común en México es el robo.<br><br>
  • El año con más crímenes durante el periodo 2015 – 2025 fue 2023.<br><br>
  • En la Ciudad de México, el número promedio de delitos más alto es el de violencia familiar, seguido de fraude y amenazas.<br><br>
  • Los delitos más frecuentes en los últimos 5 años fueron violencia familiar, otros delitos del fuero común, robo sin violencia, daño a la propiedad y amenazas.
